# GlowUP
GlowUP is an online fashion e-commerce platform designed for users in the Middle East. It provides a seamless shopping experience for men, women, and kids, allowing users to explore clothing, create outfits, and purchase items easily.  The platform focuses on user-friendly design, interactivity, and personalized fashion experiences.

## Overview
### GlowUP is a full-stack web application that enables users to:
- Browse a wide catalog of fashion products
- Create accounts and manage personal profiles
- Add items to a shopping cart
- Complete purchases through an integrated checkout system
- Discover outfit ideas and style combinations

## Features
### User Features:
- User registration and login system
- Password reset functionality
- Personalized welcome message
- Profile management

### Shopping Experience:
- Product catalog with: Images, Descriptions, Sizes & colors, Prices.
- Add to cart functionality
- Shopping bag (cart management)
- Checkout with multiple payment options

### Interactivity:
- Dynamic browsing and filtering
- Real-time cart updates
- Smooth navigation between pages

## How It Works
### User Registration/Login:
- Users create an account or log in
- A welcome message appears after login
### Browse Products:
- Users explore categories (Men, Women, Kids)
- View product details
### Add to Cart:
- Select size, color, quantity
- Items are stored in the shopping bag
### Checkout:
- Users proceed to payment
- Choose payment method (e.g., cash, card, Apple Pay)
### Order Completion
- Order is processed and stored in the database
